Application Overview

Welcome to the EC2 Management application, a web-based tool designed to streamline and manage your Amazon EC2 instances efficiently. With this application, you can:

  • Deploy EC2 Instances: Easily configure and deploy new EC2 instances based on your specified parameters such as image ID, instance type, memory size, disk size, and subnet.
  • Manage Existing Instances: View and manage your running EC2 instances. You can terminate instances directly from the management interface.

This application integrates with AWS services to provide real-time management capabilities, ensuring that you can handle your EC2 infrastructure with ease.

Use the navigation links to access the deployment configuration and instance management sections of the application.
